Introduction to Linked Lists

A linked lists is a data structure where each element is a node with two parts, the actual data to be stored in the list and a pointer to the next node in the list

Unlike static arrays, linked lists are not stored in a contiguous block of memory. Pointers in the node help to know the memory address of every node in memory. They are also allocated dynamically in the heap section of the memory

The first element node in the list is called the head node. A stack variable points to the head node

Every node in the list points to the next node. The last node points to nullptr, signifying the end of the list

Linked List Implementation in C++
            struct Node {
                int data; 
                Node* next; 
            }

You can equally use classes for implementation. Data type can also be changed to suit your needs.

Printing elements in list
            void Print() {
                Node* temp = head; 
                cout << "List is: "; 
                while(temp != nullptr) {
                    cout << " " << temp->data; 
                    temp = temp->next; 
                }
            }

A stack variable called temp is used to store the adderss of the first node or the head node. While loop is used to traverse the entire list by changing temp to point the next node at each iteration. Before changing it, the data part is printed

Inserting a node at beginning
            void Insert_Begin(int x) {
                Node* temp = new Node; 
                temp->data = x; 
                temp->next = head; 
                head = temp;  
            }

A new node is created and it's memory address is stored in a stack variable, temp. The actual data to be stored is assigned to temp->data. temp->next is set to point to the current first node to place the new one before it

After, head is now set to point to the newly created node, making it the first node in the list

Inserting a node at nth position
            void Insert_n(int x, int n) {
                Node* temp = new Node;
                temp->data = x; 
                temp->next = nullptr; 

                Node* temp1 = head; 
                for(int i=0; i <= n-2; i++) {
                    temp1 = temp1->next; 
                }
                temp->next = temp1->next 
                temp1->next = temp; 
            }

New node is created. temp1 is initialized with the address of the head node. The for loop traverses the list to the node before the node at nth position, i.e. (n-1)th node

It is n-2 in the for loop because counting starts from 0
